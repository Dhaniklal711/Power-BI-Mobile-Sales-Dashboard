# 📊 Mobile Sales Analysis Dashboard (Power BI)

## 🚀 Overview

This project is an interactive **Mobile Sales Dashboard** created using Power BI.
It helps analyze sales performance across different cities, brands, mobile models, and payment methods.

The dashboard is designed to make data easy to understand and support better business decisions.

## 🎯 Key Highlights

* 📍 **Sales by City** – Map view showing sales distribution across locations
* 📈 **Monthly Trends** – Track total quantity month-wise
* ⭐ **Customer Ratings** – Understand customer satisfaction
* 💳 **Payment Analysis** – UPI, Cards, and Cash comparison
* 📱 **Mobile Model Performance** – Top-selling models
* 📅 **Day-wise Sales** – Identify best sales days
* 🏷️ **Brand Performance** – Compare brands with sales & transactions

## 🛠️ Tools Used

* Power BI
* Power Query (Data Cleaning)
* DAX (Calculations)
* Excel Dataset

---

## 🔄 Data Processing

* Cleaned and transformed raw data
* Created custom date column
* Handled errors and missing values
* Built measures using DAX

---

## 💡 Insights

* Major sales come from big cities like Mumbai & Delhi
* Digital payments (UPI, Cards) are widely used
* Certain mobile models generate higher revenue
* Sales vary across months and weekdays
